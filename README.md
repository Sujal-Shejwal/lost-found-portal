🚀 Lost & Found Portal

🔗 Live Demo: https://lost-found-portal-bay.vercel.app/

A full-stack Lost & Found web application built using React and Firebase that allows users to report lost items, post found items, and manage claims efficiently.

This project is designed to simplify the process of reconnecting people with their lost belongings through a clean and user-friendly interface.

📌 Features

🔐 User Authentication (Login / Signup)

📦 Post Lost Items

📦 Post Found Items

📝 Claim Management System

🔔 Notifications Section

📊 User Dashboard

📱 Responsive UI Design

🔥 Real-time Database (Firebase)

🛠️ Tech Stack

Frontend:

React.js

React Router

Context API

CSS

Backend & Services:

Firebase Authentication

Firebase Firestore Database

Firebase Hosting (optional)

Vercel (Deployment)
